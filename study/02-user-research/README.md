# Tháng 2: User Research & Data Analysis

## Mục Tiêu

Học cách hiểu users thực sự — không phải assume, mà nghe và observe. Kết hợp với data analysis để đưa ra decisions dựa trên evidence.

---

## Checklist 4 Tuần

### Tuần 1-2: User Research Practice
- [ ] Đọc `01-user-research-methodologies.md`
- [ ] Đọc `02-user-interview-guide.md`
- [ ] Chọn 1 problem domain để research
- [ ] Recruit 5 người tham gia interview
- [ ] Chuẩn bị interview script (dùng template)
- [ ] Conduct 3 interviews đầu tiên
- [ ] Synthesize findings từ 3 interviews

### Tuần 3-4: Data Analysis
- [ ] Đọc `03-user-stories-acceptance-criteria.md`
- [ ] Đọc `04-sql-for-product-analytics.md`
- [ ] Conduct 2 interviews còn lại
- [ ] Tìm public dataset trên Kaggle
- [ ] Complete data analysis project
- [ ] Viết 1-page insights report
- [ ] Hoàn thành 20 SQL queries (LeetCode/Mode Analytics)
- [ ] Chọn side project idea
