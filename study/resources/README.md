# Tài Nguyên Học Tập PM

## Triết Lý Học PM

Học PM không chỉ là đọc sách và lấy chứng chỉ. PM là nghề kết hợp **kiến thức** (20%) + **kỹ năng thực hành** (40%) + **judgment từ kinh nghiệm** (40%).

**Learning path hiệu quả:**
1. Học concepts → Hiểu framework
2. Apply ngay vào dự án hiện tại → Build intuition
3. Reflect + document → Crystallize learning
4. Share với người khác → Deepen understanding

## Quick Start: Bắt Đầu Từ Đâu?

### Nếu bạn mới hoàn toàn với PM:
1. Đọc **Scrum Guide** (13 trang, miễn phí) — [scrumguides.org](https://scrumguides.org)
2. Xem **YouTube: Mike Clayton** — PM basics series
3. Tham gia **r/projectmanagement** trên Reddit
4. Hoàn thành **Bài Tập 5: PM Week Simulation**

### Nếu bạn đang develop → PM path:
1. Thi **PSM I** (Scrum.org, $150, không hết hạn) — khai thác ngay background dev
2. Học **Google PM Certificate** trên Coursera
3. Apply vào role **Scrum Master** hoặc **Technical PM** trước
4. Attend **PMI Vietnam Chapter** event gần nhất

### Nếu bạn đang ở Junior PM muốn lên mid-level:
1. Lấy **PMP** — đây là bước nhảy lương lớn nhất
2. Đọc **Making Things Happen** (Scott Berkun)
3. Build PM portfolio với 3-5 case studies
4. Mentor 1 developer junior về project planning
